Bienvenue

Cette page gère les collections Albert (Etalab) et déclenche les pipelines RAG. Tout se passe côté navigateur : la clé Albert est stockée dans le localStorage de ton navigateur, jamais transmise à un serveur tiers.

  1. Colle ta clé Albert dans le champ en haut à droite et clique Enregistrer.
  2. La clé est mémorisée localement. Recharge la page : elle est encore là.
  3. Utilise Changer pour la remplacer, Vider stockage local pour tout effacer.

Où trouver ma clé ? Sur albert.api.etalab.gouv.fr, crée un compte et génère une clé (elle commence par sk-...).
